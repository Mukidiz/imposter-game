# מתחזה 🕵️

משחק מסיבה "מתחזה" (Impostor) — למכשיר אחד שעובר ביד (pass-and-play).
סטטי לגמרי, מתארח ב-GitHub Pages.

## איך משחקים
1. בוחרים מספר שחקנים.
2. בכל סיבוב כל שחקן רואה בתורו מילה — אחד מהם מקבל **"מתחזה"** במקום המילה האמיתית.
3. עוברים בתור, וכל אחד אומר **מילה אחת** שקשורה למילה שקיבל.
4. המתחזה מנסה להשתלב ולנחש על מה מדברים; השאר מנסים לתפוס אותו.
5. לוחצים **"סיבוב הבא"** כשהקבוצה תפסה את המתחזה.

16 מילים = 16 סיבובים, בסדר קבוע (מתחיל קליל ומתחמם).

## פיתוח / הרצה מקומית
אין צורך ב-build. פשוט פותחים את `index.html` בדפדפן, או:

```bash
python3 -m http.server 8000
# ואז http://localhost:8000
```

## קבצים
- `index.html` — מבנה המסכים
- `styles.css` — עיצוב "ירוק נועז"
- `app.js` — לוגיקת המשחק
